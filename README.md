# Небольшое одностраничное приложение, которое выводит список товаров

## Стек технологий проекта

Vite + React + Typescript + CSS Modules

Для качества кода был использован pre-commit hook husky + lint-staged с настройками prettier и eslint, который форматирует код перед коммитом

## Реализованный функционал

- Вывод карточек товаров на главной странице
- Поиск, производящий фильтрацию карточек по названию товара независимо от регистра
- Модальное окно, которое появлятся при клике на карточку и содержит более подробную информацию о товаре
- Данные можно получить из имитированного API-запроса с искусственной задержкой 1 секунда
- Для состояния загрузки/ошибки запроса созданы компоненты обработчики
- Верстка адаптирована под мобильные устройства, планшеты и десктоп

Установка зависимостей

```bash
pnpm install
```

Локальный запуск проекта

```bash
pnpm dev
```
